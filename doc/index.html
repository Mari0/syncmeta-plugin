Syncmeta-Plugin

A small plugin to listen to events and create node or edges and edit their attributes. For a example how to use the demo have a look at the demo widget in the demo folder. The plugin doesn't have any dependencies.

Installation of the demo

Install the dependencies of the demo and start a HTTP server. The easiest and featest way is to use http-server.

$ bower install
